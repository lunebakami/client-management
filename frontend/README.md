# Client Management Frontend

Este é um exemplo básico de um aplicativo criado com Vite e React.

## Pré-requisitos

Antes de começar, você precisará ter o [Node.js](https://nodejs.org/) e o [npm](https://www.npmjs.com/) (ou [Yarn](https://yarnpkg.com/)) instalados na sua máquina.

## Instalação

Siga as etapas abaixo para instalar e executar o aplicativo:

1. Clone este repositório:

git clone https://github.com/lunebakami/client-management.git

2. Navegue até o diretório do projeto:

cd client-management/frontend

3. Instale as dependências:

Com npm:

npm install

Com Yarn:

yarn

4. Inicie o servidor de desenvolvimento:

Com npm:

npm run dev

Com Yarn:

yarn dev


5. Abra seu navegador e acesse [http://localhost:5173](http://localhost:5173) para visualizar o aplicativo.

## Comandos Disponíveis

- `npm run dev` ou `yarn dev`: Inicia o servidor de desenvolvimento.
- `npm run build` ou `yarn build`: Gera uma versão de produção do aplicativo na pasta `dist`.
